# Calculator made in x64-86 Assembly.

## Screenshot
![screenshot](https://github.com/Hrpav/Assembly-Calculator/blob/main/screenshot.jpg)

## Features
### Operators
- Division (/)
- Multiplication (*)
- Subtraction (-)
- Addition (+)
- Decimal (.)
- All of the numbers
### Sounds
- Plays `sounds/ding.wav` when press a button.
- Plays `sounds/err.wav` when there is an error, for example dividing by 0.
### Font
The old terminal font (`fonts/VGAOEM.FON`)

## Building the app

To build the app, simply run:

```bash
build.bat
```

Note: you should run that using the `x64 Native Tools Command Prompt for VS 2019`.
